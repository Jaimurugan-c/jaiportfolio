## Getting Started

Install dependencies:

```bash
npm install
```

Run dev server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview production build:

```bash
npm run preview
```

## Customize

- Update your info, socials, experience, and projects:
  - `src/data/content.js`
- Replace CV (keep the same filename):
  - `public/Jai_Murugan_C_CV.pdf`
- Replace profile image:
  - `public/profile.svg`
